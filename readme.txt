This plugin integrates Lytics Topics into your WordPress site.

== Description ==

*Harness the power of first-party data to understand what your customers really want with Lytics, the customer data platform built for marketers.*

You will need to sign up for a Lytics account at [www.lytics.com](https://www.lytics.com)

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `lytics-topics` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Access Lytics Topics by navigating to Posts -> Edit Lytics Topics
4. Create Lytics Topics
5. Navigate to a post and find created Lytics Topics metabox in the sidebar. Check desired Lytics Topics and update post
6. View post, right click page and click View Source. CMD + F `lytics:topics` to view selected Lytics Topics


== Frequently Asked Questions ==

= Does your plugin work with my theme? =

This plugin works with any theme provided that it uses the built in categories functionality.
